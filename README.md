## 关于函数式接口

1、如果一个接口只有一个抽象方法，那么该接口就是一个函数式接口。

2、如果我们在某个接口上声明了FunctionalInterface注解，那么编译器就会按照函数式接口的定义要求该接口。

3、如果某个接口只有一个抽象方法，但我们并没有给该接口声明FunctionalInterface注解，那么编译器依旧会将该接口看做是函数式接口。
